# Документация по настройке Windows 10

Репозиторий содержит материалы по подготовке документации для настройки Windows 10.

Состав проекта:
- структура документа;
- описание используемых стилей;
- план скриншотов;
- инструкции по оформлению.
Примеры:
git add .
git commit -m "Создание структуры проекта"

git commit -m "Заполнен README"

git commit -m "Добавлены теоретические материалы"

git commit -m "Добавлен чек-лист документа"

git commit -m "Финальная правка"
